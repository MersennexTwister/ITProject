Итак, как работает моя часть проекта.
Вся работа идёт в классе System (который, кстати, во избежание нелепых совпадений с системными файлами питона хорошо
бы переименовать). В нём 3 функции:

startWork() - начинает работу (перезагрузка системы). Нужно запускать после изменения директории с учениками.

recogniteTheFace() - делает скрин с камеры и распознаёт лица на нём. Результат передаётся в функцию getGoodFace(),
которая среди лиц находит наиболее близкое к центру.

Ещё 2 вещи:

	У класса System есть параметр - workPath. Это путь к директории с лицами.
	
	Есть функция clearFile(), которая очищает файл с енкодингами после каждого перезапуска системы, дабы не
	захламлять память.

Если что-то не нравится/не работает/надо добавить - пиши!
